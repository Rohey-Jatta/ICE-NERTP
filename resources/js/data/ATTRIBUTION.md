# gambiaAdm1.json — boundary attribution

The Gambia ADM1 (Local Government Area) boundaries in `gambiaAdm1.json` are
derived from **geoBoundaries** (gbOpen), boundary ID `GMB-ADM1-31243875`.

- Source: Knippenberg, Erwin & Moritz, Meyer — *The Hidden Potential of Mobile
  Phone Data: Insights on COVID-19 in The Gambia* (World Bank Data Catalog).
- Distributed via geoBoundaries: https://www.geoboundaries.org
- License: **Creative Commons Attribution 4.0 (CC BY 4.0)**

The file has been minified and stripped to a single `name` property per feature.
Attribution is surfaced in the public map UI (legend footer).
